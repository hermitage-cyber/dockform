DOCKFORM PORTABLE

Что это
=======
Программа для автоматического заполнения юридических документов из
шаблонов. Запускается с флешки или из любой папки на компьютере.
Установка не нужна.

Как запустить
=============
В этой папке два ярлыка:

  «Претензионная.lnk»  — работа с претензиями
  «Документация.lnk»   — документация к закупочным процедурам

Кликните по нужному. Откроется окно программы. Дальше — выбор шаблона
(или анкета для претензий), заполнение полей, кнопка «Сформировать
документ» → готовый .docx.

Если ярлыки случайно удалились — запустить можно сам файл dockform.exe.
Без аргументов он покажет экран выбора режима.

Если ярлык не открывается
=========================
Чаще всего причина — у Windows не хватает компонента WebView2
(браузерный рантайм). При первом запуске программа должна сама
предложить его скачать.

Если автоматически не получилось — скачайте установщик с сайта Microsoft:
  https://developer.microsoft.com/microsoft-edge/webview2/
Выберите «Evergreen Standalone Installer» под x64 и запустите.

Где появляется готовый документ
================================
В папке, которую вы выберете в диалоге «Сохранить как…». Для удобства
рядом с программой лежит пустая папка «output/» — можно складывать
туда.

Шаблоны и обновления
====================
Программа подтягивает свежие шаблоны с интернета при запуске (если
интернет есть). Без интернета работает с теми, что уже лежат локально
в папке templates/.

Обновление самой программы — не автоматическое. Если придёт новая
версия — администратор пришлёт новую папку (или попросит обновить
содержимое существующей).

Если не помогло
================
Напишите администратору программы, опишите шаг, на котором возникла
проблема, и приложите скриншот окна с ошибкой.
